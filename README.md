Projeto para a cadeira de Programação Orientada a Obejtos do professor Pablo Roberto.

Marco Antônio, Samuel e Diogo são as pessoas por trás desse desenvolvimento, vamos desenvolver um programa Java com Swing sobre cartão fidelidade, onde o cliente ganha pontos
quando gasta uma certa quantidade de dinheiro em um comércio, podendo trocar seus pontos por produtos ou desconto.
